## Face Video Generation Project With Flow Matching
Trained an unconditional flow matching model from scratch to generate 16 frame videos of faces. It was trained on the CelebVQ dataset of around 35K celebrity faces (https://celebv-hq.github.io/). Generated videos aren't of the best quality, but demonstrate learned structure and temporal coherence. The next step would be to do conditional training/generation based off emotion.
